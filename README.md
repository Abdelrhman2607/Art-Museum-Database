## How to Run
To run the program, execute the main Python file:

```
python code/main.py
```

Alternatively, if you're already in the `code/` directory:

```
python main.py
```

## Use of AI
AI was used to assist the development of the following parts of the program:
- Generating sample data in `db_init.sql`
- Developing the `display_query_result` method to display a nicely formatted table for the results of SQL queries

## Organization:
- `code/` folder: contains main python application code (`main.py`)
- `sql/` folder: contains all sql scripts required (database creation and initialization in `db_init.sql`, query tasks in `db_queries.sql`)
- `design/` folder: contains EERD (`eerd.drawio`) and relational schema (`rd.drawio`)
